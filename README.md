# Projeto DevWeb — Na Trave e no Topo

Este projeto é um site informativo focado em futebol brasileiro e internacional, trazendo conteúdos como história, séries A e B, Copa do Mundo, curiosidades, notícias e informações de contato.  
Ele foi desenvolvido como parte de um projeto de Desenvolvimento Web, utilizando **HTML**, **CSS** e **JavaScript**, com organização modular e design responsivo.

## Tecnologias Utilizadas

- **HTML5** – Estrutura das páginas  
- **CSS3** – Estilização, responsividade e identidade visual  
- **JavaScript** – Interações, lógica, organização dinâmica  
- **Figma** (opcional) – Para criação dos wireframes  
- **Git & GitHub** – Controle de versão e hospedagem do código  

---

## Páginas do Projeto

As páginas estão na pasta `/pages` e incluem:

- `index.html` — Página inicial  
- `noticias.html` — Notícias sobre futebol  
- `historia.html` — História do futebol brasileiro  
- `seriea.html` — Informações da Série A  
- `serieb.html` — Informações da Série B  
- `copamundo.html` — Conteúdos sobre a Copa do Mundo  
- `contato.html` — Página de contato com formulário
